# 🤖 AI Hub

AI Hub is a simple and clean website that provides information about various Artificial Intelligence tools and their specific uses.

This project was created as a microproject using HTML and CSS.

---

## 🌐 Project Objective

The main objective of this project is to help users understand different AI tools and how they can be used for specific tasks like:

- Coding
- Designing
- Writing
- Learning
- Content creation

Many users get confused about which AI tool to use. This website solves that problem by organizing tools in a structured way.

---

## 🧠 AI Tools Included

The website includes information about:

- ChatGPT  
- Gemini  
- Claude  
- Gamma  
- Canva  
- Grammarly  
- GitHub Copilot  
- Cursor  
- Leonardo AI  

Each tool has its own dedicated page with:
- About  
- Features  
- Uses  

---

## 🧱 Website Structure

The website consists of multiple pages:

- **Home Page** → Introduction + categories + featured tools  
- **Tools Page** → Displays all AI tools  
- **AI Pages** → Individual pages for each AI tool  
- **About Page** → Project details and team members  
- **Contact Page** → Contact information  

---
